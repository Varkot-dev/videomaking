### Verified ManimGL 3D API (all read from source, not assumed)

```python
# Camera — reorient takes DEGREES directly
self.frame.reorient(-30, 70)                      # theta_deg, phi_deg
self.frame.add_ambient_rotation(angular_speed=0.2) # kwarg is angular_speed NOT speed
self.frame.clear_updaters()                        # stop rotation before FadeOut

# ThreeDAxes
axes = ThreeDAxes(x_range=[-3,3,1], y_range=[-3,3,1], z_range=[-2,2,1])

# ParametricSurface — uv_func returns raw np.array, NOT axes.c2p()
surface = ParametricSurface(
    lambda u, v: np.array([u, v, np.sin(u) * np.cos(v)]),
    u_range=(-PI, PI), v_range=(-PI, PI), resolution=(32, 32),
)

# GLSL coloring — takes a string, NOT a Python lambda
surface.set_color_by_xyz_func("z", min_value=-1.0, max_value=1.0, colormap="viridis")

# Shading
surface.set_shading(0.8, 0.5, 0.5)

# Mesh — add as child so it moves with surface
mesh = SurfaceMesh(surface, resolution=(12, 12))
surface.add(mesh)

# 2D text in ThreeDScene — must pin to frame
self.fix_in_frame(*[label1, label2])   # NOT add_fixed_in_frame_mobjects (doesn't exist)

# Transparency — deactivate depth test so surface doesn't occlude objects behind it
surface.set_opacity(0.08)
self.deactivate_depth_test()  # called on scene, not on mobject
```

### What does NOT exist / crashes
- `self.frame.set_euler_angles(theta * DEGREES)` — use `reorient(theta_deg, phi_deg)`
- `set_color_by_xyz_func(lambda x, y, z: z)` — takes GLSL string, not Python callable
- `axes.c2p(u, v, z)` inside ParametricSurface uv_func — uv_func returns raw world coords
- `add_ambient_rotation(speed=...)` — kwarg is `angular_speed=`
- `add_fixed_in_frame_mobjects()` — use `self.fix_in_frame(*mobjects)` instead
- `TexturedSurface` — SSL cert issue downloading textures; avoid in pipeline

### Interior reveal pattern (correct order matters)
```python
# 1. Add inner object FIRST so it's behind the outer
self.add(inner_torus)
self.add(outer_sphere)
# 2. Orbit camera — both objects rotate together because both are in scene
self.frame.add_ambient_rotation(angular_speed=0.15)
# 3. Go transparent AFTER orbit starts
self.play(outer_sphere.animate.set_opacity(0.08), run_time=2)
self.deactivate_depth_test()  # must call after going transparent
```
